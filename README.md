# hot-ramen
A simple data analysis of ramen ratings using [The Ramen Rater's dataset](https://www.theramenrater.com/resources-2/the-list/).

## Dependencies
- NumPy
- Matplotlib
- BaseMap (requires geos)

## Result
Number of Ratings: 2685
